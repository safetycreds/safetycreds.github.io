# safetycreds.github.io

Static website for [safetycreds.org](https://safetycreds.org), served by GitHub Pages.

Currently a single under-construction landing page describing the goal of the SafetyCreds application: tracking the safety behaviour of subscribing users and letting them redeem earned safety credits as insurance benefits with partnering insurance companies.

Main project documentation lives in the [safetycreds master repository](https://github.com/safetycreds/safetycreds).
